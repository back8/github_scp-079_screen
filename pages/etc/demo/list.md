## 加入本列表

如您已在运行 SCP-079 大部分实例，公开并接受其他人的使用请求，且能保证以下规范：

1. 长期运营
2. 及时跟随本项目进行更新
3. 定期进行规则维护
4. 审慎使用群组赋予机器人的权限

则您可联系 [SCP-079-TICKET](https://t.me/SCP_079_TICKET_BOT)，申请将您的服务提供方式添加到本页列表中。

如发现您的服务有滥用权限、恶意推广等现象，SCP-079 工作组将从此列表中永久移除您的服务。

申请添加至此列表时，请提供以下信息：

1. 获取服务的联系方式（频道 / 群组 / 私聊机器人 / 网站）
2. 可提供的服务（按项目名称），例如：USER、CAPTCHA、TIP
3. 服务的运行模式：独立工作 / All-In-One
4. 服务主要面向的群体
5. 一句话简述

## 交流合作

SCP-079-INSIDER 是一个提供给 SCP-079 实例搭建运营方的交流群组。SCP-079 工作组 将在 SCP-079-INSIDER 中为第三方实例搭建者提供有限的技术支持和帮助。

欢迎 SCP-079 第三方实例搭建者通过 [SCP-079-TICKET](https://t.me/SCP_079_TICKET_BOT) 申请加入 SCP-079-INSIDER。

为防止相关服务遭受滥用，加之受安全方面的因素影响，加入 SCP-079-INSIDER 需要满足以下条件：

1. 搭建并运营 SCP-079 实例。
2. 其维护的 SCP-079 实例为至少 3 个「成员数量不低于 1000 人且日均消息量不低于 500 条的群组」提供服务。

注：

- 申请加入 SCP-079-INSIDER 可能需要较长的审核时间。
- SCP-079-INSIDER 仅接受每个第三方实例的一名代表加入其中。
- SCP-079 工作组保留由 SCP-079 工作组中一名成员加入其交流群组（类似 [SCP-079-CHAT](https://t.me/SCP_079_CHAT)）的权利。
- 实际控制人切割需要预先告知，否则需要重新执行认证和核验过程（存在无法通过的可能性）。
